# FRC Team 5142 - RoboDominators 2026

Competition robot code for the 2026 FRC season.

## Features
- Phoenix 6 Swerve Drive
- QuestNav + Pigeon Hybrid Gyro
- Dual Limelight Vision Fusion
- PhotonVision Object Detection
- PathPlanner Autonomous
- AdvantageKit Logging

## Setup
1. Install WPILib 2025
2. Clone this repo
3. Open in VS Code
4. Build: `./gradlew build`
5. Deploy: `./gradlew deploy`
